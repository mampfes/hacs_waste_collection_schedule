# Ecolan - Lanciano

Ecolan - Lanciano is supported by the generic [ICS](/doc/source/ics.md) source. For all available configuration options, please refer to the source description.


## How to get the configuration arguments

Use the Google Calendar ICS feed URL below. This calendar covers the Lanciano area served by Ecolan.

## Examples

### Lanciano

```yaml
waste_collection_schedule:
  sources:
    - name: ics
      args:
        url: https://www.google.com/calendar/ical/3vthrt88quqepasgt6hkn7dkc8%40group.calendar.google.com/public/basic.ics
```
