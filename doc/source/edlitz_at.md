# Marktgemeinde Edlitz

Support for schedules provided by [Marktgemeinde Edlitz](https://edlitz.at).

Source for Marktgemeinde Edlitz, AT

## Configuration via configuration.yaml

```yaml
waste_collection_schedule:
  sources:
    - name: edlitz_at
```

### Configuration Variables

No configuration arguments are required.

## Example

```yaml
waste_collection_schedule:
  sources:
    - name: edlitz_at
```
