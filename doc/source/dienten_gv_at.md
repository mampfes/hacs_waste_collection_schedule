# Dienten am Hochkönig

Support for schedules provided by [Dienten am Hochkönig](https://www.dienten.gv.at).

Waste collection schedule for Dienten am Hochkönig, Austria.

## Configuration via configuration.yaml

```yaml
waste_collection_schedule:
  sources:
    - name: dienten_gv_at
```

### Configuration Variables

No configuration arguments are required.
