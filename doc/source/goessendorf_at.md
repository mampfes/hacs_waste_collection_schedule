# Marktgemeinde Gössendorf

Support for schedules provided by [Marktgemeinde Gössendorf](https://www.goessendorf.com/).

Source for Marktgemeinde Gössendorf, AT

## Configuration via configuration.yaml

```yaml
waste_collection_schedule:
  sources:
    - name: goessendorf_at
```

### Configuration Variables

No configuration arguments are required.

## Example

```yaml
waste_collection_schedule:
  sources:
    - name: goessendorf_at
```
