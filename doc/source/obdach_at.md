# Marktgemeinde Obdach

Support for schedules provided by [Marktgemeinde Obdach](https://www.obdach.gv.at/).

Source for Marktgemeinde Obdach, AT

## Configuration via configuration.yaml

```yaml
waste_collection_schedule:
  sources:
    - name: obdach_at
```

### Configuration Variables

No configuration arguments are required.

## Example

```yaml
waste_collection_schedule:
  sources:
    - name: obdach_at
```
