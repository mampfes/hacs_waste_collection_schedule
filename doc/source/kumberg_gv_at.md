# Kumberg

Support for schedules provided by [Kumberg](https://www.kumberg.gv.at).

Source for Kumberg.

## Configuration via configuration.yaml

```yaml
waste_collection_schedule:
  sources:
    - name: kumberg_gv_at
```

### Configuration Variables

No configuration arguments are required.

## Example

```yaml
waste_collection_schedule:
  sources:
    - name: kumberg_gv_at
```
