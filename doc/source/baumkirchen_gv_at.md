# Baumkirchen

Support for schedules provided by [Baumkirchen](https://www.baumkirchen.gv.at).

Waste collection schedule for the municipality of Baumkirchen, Austria.

## Configuration via configuration.yaml

```yaml
waste_collection_schedule:
  sources:
    - name: baumkirchen_gv_at
```

### Configuration Variables

No configuration arguments are required.

## Example

```yaml
waste_collection_schedule:
  sources:
    - name: baumkirchen_gv_at
```
