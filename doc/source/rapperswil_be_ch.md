# Rapperswil

Support for schedules provided by [Rapperswil](https://www.rapperswil-be.ch/).

Source for Rapperswil.

## Configuration via configuration.yaml

```yaml
waste_collection_schedule:
  sources:
    - name: rapperswil_be_ch
```

### Configuration Variables

No configuration arguments are required.

## Example

```yaml
waste_collection_schedule:
  sources:
    - name: rapperswil_be_ch
```
