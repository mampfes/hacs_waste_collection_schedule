# Piberbach

Support for schedules provided by [Piberbach](https://www.piberbach.ooe.gv.at).

Source for Piberbach, Austria.

## Configuration via configuration.yaml

```yaml
waste_collection_schedule:
  sources:
    - name: piberbach_ooe_gv_at
```

### Configuration Variables

No configuration arguments are required.

## Example

```yaml
waste_collection_schedule:
  sources:
    - name: piberbach_ooe_gv_at
```
