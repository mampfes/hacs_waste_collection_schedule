# Jochberg

Support for schedules provided by [Jochberg](https://www.jochberg.gv.at).

Waste collection schedule for the municipality of Jochberg, Austria.

## Configuration via configuration.yaml

```yaml
waste_collection_schedule:
  sources:
    - name: jochberg_gv_at
```

### Configuration Variables

No configuration arguments are required.

## Example

```yaml
waste_collection_schedule:
  sources:
    - name: jochberg_gv_at
```
