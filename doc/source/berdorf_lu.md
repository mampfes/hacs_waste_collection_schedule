# Gemeng Bäertref

Support for schedules provided by [Gemeng Bäertref](https://www.berdorf.lu).

Source for Berdorf commune (Gemeng Bäertref) waste collection schedule, Luxembourg.

## Configuration via configuration.yaml

```yaml
waste_collection_schedule:
  sources:
    - name: berdorf_lu
```

### Configuration Variables

No configuration arguments are required.

## Example

```yaml
waste_collection_schedule:
  sources:
    - name: berdorf_lu
```
